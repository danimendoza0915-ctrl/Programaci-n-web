# Sitio web del sistema operativo

# Descripcion del sistema
Este para darle solucuión al caso de estudio el cual trata
de poder poder ofrecer un sitio web informativo
institucional.

# Objetivo
El objetivo con este sistema es poder mostrar y difundir
la información de la institución, los servicios que ofrece
la institución y los datos de contacto.

# Tecnologias utilizada
para poder llevar acabo el diseño del sistema primeramnete 
se es el uso de HTML para la estructura del sitio web y 
a su vez el uso de CSS para poder dar el diseño al sitio 
web.
